# Chord Lagu Sion

Salam semuanya!

Kali ini saya memilih untuk menggunakan GitHub sebagai tempat penyimpanan karena memiliki beberapa kelebihan:
* Ada history edit file
* Mudah untuk dicopy dan didownload
* Mudah untuk kolaborasi
